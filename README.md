## Pomodoro Timer

Built using React with moments.js library

Simple web app that allows users to set and keep track of custom Pomodoro Timers.

### To run the application

```bash
$ git clone https://github.com/barclayd/PomodoroTimer
$ cd PomodoroTimer
$ npm install
$ npm start
```
